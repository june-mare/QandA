## ゲームとして工夫した点
- セーブとロードを出来るようにし、続きからゲームが開始できます。
- Logの確認を出来るようにしました。
- テキストを一文字ずつ描画してしっかりと登場人物が話しているように見せるために工夫しました。

## プログラムとして工夫した点
- 構造体を使いシーンごとのテキストやキャラクターを管理しやすくしました。 
- 関数に挑戦しメインループ自体を見やすくしました。
- 文字送りやクロマキーの処理などWindowsAPIの機能を理解応用した機能を入れました。
